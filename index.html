УСЛОВИЯ УЧАСТИЯ

  • Розыгрыш проводится среди подписчиков нашего Telegram-канала
  • Для участия необходимо быть подписанным на наш канал и бот
  • Каждый участник может получить только один подарок
  • Розыгрыш проводится с помощью подсчета баллов бота каждого пользователя
  • Победитель будет объявлен в нашем Telegram-канале
  • Для получения подарка победитель должен связаться с нами в течение 2х часов
  • Организатор оставляет за собой право изменить условия акции

КАК ЗАРАБАТЫВАТЬ БАЛЛЫ?

  • Приглашать друзей в канал с помощью личной реферальной ссылки. Ее можно найти в боте!

КАК МЫ СЧИТАЕМ БАЛЛЫ?

  • ВСЕ баллы считаются по формуле: (Количество рефералов * 20) + (Количество опросов * 5)
